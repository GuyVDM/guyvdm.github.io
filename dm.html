DeviousMUD Clone

DeviousMUD Clone is a passion project of mine, reflecting my favorite game of all time: "Runescape".

It is done entirely by myself in C++. My game development journey started at the age of 12, when I worked on a personal Runescape Private server made completely in Java. In similar fashion 12 years later with more experience in programming, I'm working on creating my own interpretation of the game.

World Editor

The most recent item I've been working on for the project is a World Level Editor tool, this tool is aimed to be used by close friends of mine so they can contribute to the game by creating their own segments of the world.

This tool is made using Imgui docking branch and sdl, the goal of this world editor engine is to be able to export chunk data that can be interpreted both by the client and server mainly for server sided pathing, client/server interaction sanity checks, chunk data for optimization and more.

As of writing this the tool has been in development for just a couple of days.
